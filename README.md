# WEATHER-APP

A simple and elegant weather application built with Python and Tkinter that fetches real-time weather data from OpenWeatherMap API.

## Features

- **Real-time weather data**: Get current weather information for any city
- **User-friendly interface**: Clean and intuitive GUI built with Tkinter
- **Multiple weather metrics**: Temperature, humidity, wind speed, and weather description
- **Dynamic background**: Background color changes based on weather conditions
- **Error handling**: Graceful handling of network errors and invalid city names
